# Lens Studio Runner - Endless Runner Game

Lens Studio Runner is a simple endless-runner game built in Snap Lens Studio.
The player controls a Bitmoji character running forward, dodging obstacles, switching lanes, and collecting coins.
The project demonstrates how to implement core gameplay mechanics inside an AR lens environment.

- Global EventBus system
